🧩 Cocos Creator Build Extensions

Thư viện mở rộng cho hệ thống build của Cocos Creator. Hỗ trợ tùy biến quy trình build đa nền tảng.

Tính năng chính

  • ✔️ Hook tiền/ hậu build (beforeBuild, onAfterBuild...)
  • ✔️ Custom template cho Android / iOS native
  • ✔️ Giao diện cấu hình build trong Extension Panel
  • ✔️ Tự động sửa file Gradle, manifest, plist, .mm
  • ✔️ Tương thích Cocos Creator 3.x

Cấu trúc thư mục


recreate-uuid/
├── extension.json
├── main.ts
├── hooks/
│   ├── beforeBuild.ts
│   └── onAfterBuild.ts
├── template/
│   ├── android/
│   └── ios/

Hướng dẫn sử dụng

  1. Copy thư mục `recreate-uuid` vào thư mục `extensions` của Cocos project.
  2. Khởi động lại Cocos Creator → Extension sẽ được load.
  3. Cấu hình thông qua menu: Extension → Build Config.

Ghi chú thêm

Dự án được thiết kế để mở rộng dần, cho phép người dùng thêm logic tùy biến cho từng platform.
