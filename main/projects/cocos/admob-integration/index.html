📱 Google AdMob Android Integration

Tiện ích mở rộng Cocos Creator v2.x.x giúp tích hợp Google AdMob cho nền tảng Android: tự động cấu hình AndroidManifest.xml, AppActivity.java, Gradle, và Resource.

✨ Tính năng chính

  • ✔️ Thêm APPLICATION_ID vào AndroidManifest.xml tự động
  • ✔️ Thêm ID Banner, Interstitial, Rewarded, AppOpen, Native, RewardedInterstitial vào strings.xml
  • ✔️ Chèn mã khởi tạo và huỷ trong AppActivity.java
  • ✔️ Cấu hình Gradle tự động
  • ✔️ Tùy chọn bật/tắt chỉnh sửa AppActivity
  • ✔️ Hỗ trợ Injection Java (thư mục template injection)
  • ✔️ Hoạt động với Cocos Creator v2.x.x

🗂️ Cấu trúc thư mục


google-admob-android/
├── extension.json
├── main.js
├── android-constants.js
├── config-manager.js
├── utils/
│   └── file-utils.js
├── template/
│   ├── android/
│   │   ├── injection/
│   │   ├── java/
│   │   └── gradle/

🚀 Hướng dẫn sử dụng

  1. Copy thư mục google-admob-android vào thư mục extensions của dự án Cocos Creator.
  2. Khởi động lại Cocos Creator để extension được load.
  3. Vào Extension → Google AdMob Android.
  4. Nhập Application ID và các Ad Unit ID trong bảng config.
  5. Build Android → Extension tự động chèn cấu hình cần thiết.
  6. Kiểm tra AndroidManifest.xml, AppActivity.java, Gradle, Resource để xác nhận.
